Include ZIO Actors in your project by adding the following to your `build.sbt`:

```scala
libraryDependencies += "dev.zio" %% "zio-actors" % "0.0.1+10-51dde802"
```

## Learning More

To learn more about ZIO Actors, see the [Overview](overview.md).
